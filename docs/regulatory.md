# Regulatory Compliance Notes

## CPF Rate Changes

- **2026**: OW ceiling increases from S$7,400 to S$8,000. Age bands 55-60 and 60-65 see +0.5% employer and +1% employee.
- Source: cpf.gov.sg

## Annual Maintenance

- Every January: check Budget announcement for CPF changes
- October-November: download new rate tables from CPF Board
- December: run full test suite with new rates
- 1 January: verify first pay run of year
- February: generate IR8A, submit to IRAS by 1 March
