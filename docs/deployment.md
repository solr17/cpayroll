# Deployment Guide

## Infrastructure
- **Database**: Neon (neon.tech) — free tier for dev, Pro for production
- **Redis**: Upstash (upstash.com) — free tier for dev
- **Hosting**: Hetzner VPS, Railway, or Render

## Environment Variables
See `.env.example` for required variables.

## Build & Deploy
```bash
npm run build
npm run start
```
